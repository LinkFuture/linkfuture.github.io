# UniBucket: Universal Multi-Cloud Object Storage Manager

Manage all your cloud buckets from one beautiful, cross-platform desktop application. UniBucket lets you control AWS S3, Google Cloud Storage, and any S3-compatible service seamlessly from a single unified interface. 

---

## 🌟 Why UniBucket?

UniBucket is designed for developers, teams, and anyone who needs a reliable, user-friendly, and powerful way to handle objects across multiple cloud providers. With local security, background task management, and an elegant UI, it simplifies deep folder navigation and robust file transfers.

### Supported Providers
- **AWS S3**
- **Google Cloud Storage**
- **Any S3-Compatible Service** (e.g., MinIO, DigitalOcean Spaces, Cloudflare R2, Wasabi, etc.)

---

## ✨ Key Features

- **Multi-Cloud Support**: Connect to AWS S3, Google Cloud Storage, and other S3-compatible services seamlessly. Switch between providers without friction.
- **Visual File Browser**: Navigate your cloud storage intuitively. Toggle between Grid and List views. Breadcrumb navigation and pagination make traversing deep directory trees simple.
- **Built-in Smart Previewer**: Directly preview a vast array of common formats inside the app without needing to download them first:
  - Images
  - PDFs
  - Word documents
  - Spreadsheets
  - Text & Code files
- **Reliable Transfers**: Background task management ensures your uploads and downloads complete safely and reliably—even for massive files. 
- **Dark & Light Mode**: A beautifully polished UI that respects your system's theme preference, with an option to manually configure light or dark mode.
- **Secure Credentials**: Your cloud credentials are encrypted and stored safely on your local machine. Nothing is ever sent to third-party servers.
- **Integrated Code Editor**: Powered by the same technology behind VS Code (Monaco). Features rich IntelliSense, advanced validation, and basic syntax colorization for editing files on the fly.
- **Markdown Editor & Previewer**: Write, edit, and preview Markdown documents in real-time without leaving your storage workflow.
- **Batch Metadata Editor**: Effortlessly update and manage file metadata in bulk, helping you organize large datasets efficiently.
- **Complete Object Organization**: Create folders, dedicated buckets, and manage the full lifecycle of files, including delete and restore capabilities.
