Features

Everything You Need in One Place

A unified toolkit for cloud storage management — designed for developers and teams.

Multi-Cloud Support

Connect AWS S3, Google Cloud Storage, and any S3-compatible service. Switch between providers without friction.

Visual File Browser

Toggle between grid and list views. Breadcrumb navigation and pagination make deep directory traversal simple.

Built-in Previewer

Preview images, PDFs, text files, code, Word docs, and spreadsheets directly inside the app — no downloads needed.

Reliable Transfers

Background task management ensures your uploads and downloads complete reliably — even for large files.

Dark & Light Mode

Full dark and light theme support. The app follows your system preference — or you can manually override it.

Secure Credentials

Your cloud credentials are securely stored locally on your machine. Nothing is sent to third-party servers.

Integrated Code Editor

Powered by the technology behind VS Code, featuring IntelliSense, advanced validation, and basic syntax colorization.

Markdown Editor & Previewer

Write, edit, and preview Markdown documents in real-time without leaving your storage workflow.

Batch Metadata Editor

Effortlessly update and manage file metadata in bulk to organize large datasets efficiently.
